### Какой кейс решаем?
Написать программу, которая угадывает загаданное комьютнром число за минимальное количество попыток.    

**Условия соревнования:**
- Компьютер загадывает целое число от 0 до 100, нужно его "угадать", то есть написать программу, угадывающую число. 
- Алгоритм учитывает информацию, больше или меньше случайное число, чем нужное нам число.

**Метрика качества**
Результаты оцениваются по среднему количеству попыток при 1000 повторениях. Необходимо добиться минимального количества попыток.

**Что практикуем**
Учимся писать хороший код на Python
